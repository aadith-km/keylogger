# Keyboard Event Monitoring Demo

## Overview

This project demonstrates keyboard event handling in Python using the `pynput` library. It is designed as a simple educational example to show how keyboard listeners and file operations work in Python.

## Features

* Listens for keyboard events.
* Demonstrates callback functions.
* Writes captured events to a text file.
* Simple and beginner-friendly Python project.

## Requirements

* Python 3.x
* pynput

Install the required package:

```bash
pip install pynput
```

## How It Works

1. Starts a keyboard listener.
2. Detects key press events.
3. Processes each event through a callback function.
4. Stores the detected key information in a text file.

## Learning Objectives

This project helps beginners understand:

* Event-driven programming
* Python file handling
* Keyboard event listeners
* Callback functions
* Basic cybersecurity concepts related to input monitoring
